# 0x07. Networking basics #0
This project covers the basics of computer networks and how they work as part of the ALX SE programme task: [0x07. Networking basics #0](https://alx-intranet.hbtn.io/projects/259). We touch on what is OSI Model, LAN, WAN, Internet, IP address, and TCP/UDP.
